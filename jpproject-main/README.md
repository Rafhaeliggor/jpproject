# Passo a passo para usar o projeto

Siga as instruções abaixo para clonar, instalar as dependências e executar o projeto localmente.

---

## Pré-requisitos

- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) (geralmente já vem com o Node.js)
- [Git](https://git-scm.com/) (para clonar o repositório)

---

## 1. Clone o repositório

Abra o terminal e execute:

```bash
git clone https://github.com/Rafhaeliggor/jpproject
```

---

## 2. Instale as dependências

Dentro da pasta do projeto, rode:

```bash
npm install
```

---

## 3. Execute o servidor de desenvolvimento

```bash
npm run dev
```

---

## 4. Acesse a aplicação

Abra o navegador e visite:

```
http://localhost:3000
```

Você verá a tela principal do projeto (prova de conceito do design colaborativo de viagem).

---

## 5. Interromper o servidor

Pressione `Ctrl + C` no terminal onde o servidor está rodando.

---

## Observação

Não é necessário configurar banco de dados ou variáveis de ambiente. Todos os dados são mockados e o projeto está pronto para uso imediato.
